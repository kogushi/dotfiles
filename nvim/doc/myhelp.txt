*myhelp.txt*    My help file for NeoVim

INTRODUCTION                                         *myhelp-intro*
My help file to use NeoVim.

COMMANDS                                             *myhelp-commands*
- :helptags <path>  Update tags for help.

KEY MAPPINGS                                         *myhelp-keymappings*
- <leader>h         Shortcut for showing helper.

Terminal
- <Leader>t         Open Terminal
- <c-\><c-n>        Into Terminal-Normal mode.
- i                 Return to terminal (insert mode).

quickhl-vim
- <leader>m         Hightlight a word.
- <leader>M         Reset hightlight.
